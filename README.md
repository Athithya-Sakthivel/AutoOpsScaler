# AutoOpsScaler
### ⚠️ If you are using WSL or If you're not using native Linux follow these steps
#### These setup are essential to prevent any issues later on (Download these stable and syncing versions)

---

####  For Windows

1. **Install prerequisite (VC++ 2015–2022 Runtime):**  
   [Download vc_redist.x64.exe](https://aka.ms/vs/17/release/vc_redist.x64.exe)

2. **Install VirtualBox 7.0.14:**  
   [Download VirtualBox-7.0.14-161095-Win.exe](https://download.virtualbox.org/virtualbox/7.0.14/VirtualBox-7.0.14-161095-Win.exe)

3. **Install Vagrant 2.4.3:**  
   [Download vagrant_2.4.3_windows_amd64.msi](https://releases.hashicorp.com/vagrant/2.4.3/vagrant_2.4.3_windows_amd64.msi)

4. **Restart your PC**

---

####  For macOS

1. **Install VirtualBox 7.0.14:**  
   [Download VirtualBox for macOS](https://download.virtualbox.org/virtualbox/7.0.14/)

2. **Install Vagrant via Homebrew (Ensure 2.4.3):**

   ```bash
   brew tap hashicorp/tap
   brew install hashicorp/tap/hashicorp-vagrant
